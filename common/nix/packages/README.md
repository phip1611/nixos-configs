# common packages

Collection of additional packages that I like to use in Nix(OS) but that are not
worth upstreaming.

## Documentation

This directory bundles some utility packages, such as

- `clion-exclude-direnv`
- `colortest`
- `ddns-update`
- `ftp-backup`
- `keep-directory-diff`
- `link-to-copy`
- `nix-shell-init`
- `normalize-file-permissions`
- `qemu-uefi`
- `run-efi`
- `strace-with-colors`
